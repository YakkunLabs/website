# Yakkun Labs Website

This is the frontend of the Yakkun Labs website built using React and TypeScript.

## How to Setup

Open terminal and go to:

D:\projects\website\client

Install all dependencies:

npm install

Start the development server:

npm start

Build for production:

npm run build

## Folder Structure

client/
  └── public/
  └── src/
      ├── assets          → all images
      ├── components      → reusable components (Header etc.)
      ├── pages           → all the pages (Home, Services, etc.)
      ├── styles          → CSS files
      ├── utils           → utility functions (if any)
      ├── hooks           → custom hooks (if any)
      └── types           → TypeScript types

## Available Pages

/             → Home  
/services     → Services  
/industries   → Pricing  
/company      → About  
/resources    → Portfolio  
/careers      → Careers  
/support      → Support

## Notes

- All styles are inside `src/styles`
- Images are stored in `src/assets`
- The website follows dark purple and cream theme
- Built using best practices with reusable components
